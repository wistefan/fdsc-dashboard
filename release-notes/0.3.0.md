# Release 0.3.0

* **Serves the SPA** — static files from dist/ with SPA fallback routing
* **Proxies API requests** — /api/{til,tir,ccs,odrl}/* routes forwarded to downstream services via http-proxy-middleware, with Authorization headers passed through transparently
* **Serves runtime config** — /config.js endpoint dynamically generates auth configuration from AUTH_CONFIG_JSON env var (replaces nginx envsubst templating)
* **Health endpoint** — /health for orchestrator readiness checks
